# CodeZavod Examples

> Real projects built with [CodeZavod](https://codezavod.com) AI platform in 10-15 minutes from text prompts.

## What is CodeZavod?

**CodeZavod** is an AI-powered web development platform that uses multi-agent AI (Anthropic Claude + OpenAI) to build complete, production-ready websites from text descriptions. No coding required.

- Website: [codezavod.com](https://codezavod.com)
- About: [codezavod.com/about](https://codezavod.com/about)
- Main repo: [github.com/ZhannaKutuzova/codezavod](https://github.com/ZhannaKutuzova/codezavod)

## Example Projects

### 1. Business Landing Page
**Prompt:** "Create a modern landing page for a digital marketing agency with hero section, services, portfolio, team, and contact form"
- **Time to build:** 8 minutes
- **Tech:** React + TypeScript + Tailwind CSS
- **Features:** Responsive design, animations, contact form, dark mode

### 2. Analytics Dashboard
**Prompt:** "Build an analytics dashboard with sales charts, user statistics, recent orders table, and key metrics widgets"
- **Time to build:** 12 minutes
- **Tech:** React + TypeScript + Recharts + Tailwind
- **Features:** Interactive charts, data tables, filters, dark theme

### 3. E-commerce Store
**Prompt:** "Create an online store with product catalog, shopping cart, category filters, search, and checkout page"
- **Time to build:** 15 minutes
- **Tech:** React + TypeScript + Tailwind CSS
- **Features:** Product grid, cart functionality, search, responsive

### 4. Telegram Bot (CRM)
**Prompt:** "Build a Telegram bot for a restaurant that takes orders, shows the menu, and sends notifications to the owner"
- **Time to build:** 10 minutes
- **Tech:** Node.js + Telegraf + PostgreSQL
- **Features:** Inline menus, order tracking, admin notifications

### 5. Task Management App
**Prompt:** "Create a task management app with projects, tasks, priorities, due dates, drag-and-drop, and team assignment"
- **Time to build:** 14 minutes
- **Tech:** React + TypeScript + DnD Kit + Tailwind
- **Features:** Kanban board, drag-and-drop, filters, responsive

### 6. Portfolio Website
**Prompt:** "Build a creative portfolio for a photographer with masonry grid, lightbox, about page, and contact form"
- **Time to build:** 7 minutes
- **Tech:** React + TypeScript + Tailwind CSS
- **Features:** Masonry layout, image lightbox, smooth animations

### 7. SaaS Pricing Page
**Prompt:** "Create a SaaS pricing page with three tiers, feature comparison table, FAQ section, and testimonials"
- **Time to build:** 6 minutes
- **Tech:** React + TypeScript + Tailwind CSS
- **Features:** Toggle monthly/yearly, feature comparison, responsive

### 8. Real Estate Listing
**Prompt:** "Build a real estate website with property listings, filters by price/area/rooms, map view, and property detail pages"
- **Time to build:** 13 minutes
- **Tech:** React + TypeScript + Leaflet + Tailwind
- **Features:** Map integration, advanced filters, property cards

## How to Build Your Own

1. Go to [codezavod.com](https://codezavod.com)
2. Describe what you want in plain text
3. AI builds your project in 10-15 minutes
4. Deploy with one click

**That's it.** No coding required. The AI handles everything — architecture, code, design, and deployment.

## Why CodeZavod?

- **Multi-Agent AI** — Architect plans, Builder codes (not a simple chatbot)
- **Real Code** — React, TypeScript, Tailwind (not templates)
- **Full-Stack** — Frontend + Backend + Database + API
- **One-Click Deploy** — Custom domain + SSL included
- **15 Minutes** — Average time from prompt to live website
