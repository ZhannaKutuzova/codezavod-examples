## How It Works

```
1. Describe your project in text
2. Router -> determines project type
3. Architect -> plans architecture, database, API
4. Builder -> writes code in WebContainer with live preview
5. Mr Fixer -> auto-fixes any errors
6. One-click deploy to Railway with custom domain
```

## Technology

- Runtime: WebContainer (Node.js in browser with HMR)
- AI: Anthropic Claude + OpenAI (4 agents)
- Database: Supabase PostgreSQL (auto-provisioned, works in preview)
- Mobile: React Native + Expo
- Video: Claude + Gemini + ElevenLabs + Kling AI + FFmpeg
- Integrations: 11 pre-built (Telegram, Slack, Discord, Stripe, etc.)
- Deploy: Railway + custom domains + SSL
